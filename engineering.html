Projetos em curso

Carro seguidor de linha

Descrição

Projeto inicial com o carro de tração diferencial que comprei. Vai utilizar-se um Arduino Uno R3 bem como fototransístores para criar um robot seguidor de linha. No final vou desenvolver vários controladores em software (PI, PID, Fuzzy) para melhorar o desempenho. Vão retirar-se conclusões do trabalho elaborado.
